## 투어 상품 예약 서버

## Installation

```bash
$ npm install
```

## Running the app

```bash
$ docker compose up -d
port는 3000번
```

## Docs

```
http://localhost:3000/docs 참고
```

## DB initialization
```
/sql/initdb.d 폴더 참고
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
